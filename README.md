# VUT FIT IZU - Project 1
Simple pathfinding algorith specificaly made for 1. project in IZU

## Compilation
```
make
```

## Running program
```
./prog < [TABLE]
```

## Table format
```
xSize   ySize
xStart  yStart
xEnd    yEnd
0x-0y Value . . . Nx-Oy Value
.                 .
.                 .
.                 .
Ox-Ny Value. . . Nx-Ny Value
```
